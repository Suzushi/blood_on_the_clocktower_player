# 🩸 血染钟楼 - 在线网游

这是一个基于 Web 的《血染钟楼》（Blood on the Clocktower）桌面游戏。它旨在提供一个线上游戏环境，让玩家能够在不依赖线下游戏环境的情况下，与其他玩家互动。

## ✨ 核心功能

- **📖 剧本支持**：内置三大官方剧本数据（暗流涌动、黯月初升、梦殒春宵）。
- **✅ 第一阶段完成**：暗流涌动（Trouble Brewing）主流程与核心角色逻辑已稳定可用。
- **🚧 第二阶段筹备**：黯月初升（Bad Moon Rising）与梦殒春宵（Sects & Violets）进入规则梳理与工程化落地阶段。
- **🤖 自动化夜间流程**：
  - 自动计算并显示正确的夜间唤醒顺序。
  - **智能信息生成**：自动计算洗衣妇、共情者、厨师、调查员等角色的查验信息，甚至考虑了**中毒/醉酒**的影响。
  - **复杂角色支持（已实装部分）**：月之子、莽夫、驱魔人、僵怖、沙巴洛斯、珀、恶魔代言人等机制已接入当前结算体系。
- **🗳️ 白天流程管理**：
  - 可视化的玩家座位表，直观显示存活/死亡状态。
  - 内置提名与投票计算器，自动判断处决结果。
  - **杀手专属 UI**：白天阶段为杀手提供专属操作界面。
  - 一键处理复活、死亡、更改状态。
- **🧩 代码结构升级**：
  - 平衡参数已独立到 `config/balance.py`。
  - 路由开始拆分至 `routes/`（首批 gameplay 蓝图已落地）。
  - 信息生成逻辑开始下沉至 `services/`，便于后续多剧本扩展。


## 🚀 快速开始

### 环境要求

- Python 3.8 或更高版本

### 安装步骤

1. **克隆或下载本项目**到本地。
2. **安装依赖库**：
    打开终端（CMD 或 PowerShell），进入项目根目录，运行：

    ```bash
    pip install -r requirements.txt
    ```

### 启动游戏

1. **运行程序**：

    ```bash
    python main.py
    ```

2. **访问界面**：
    打开浏览器，访问地址：`http://127.0.0.1:5000/player`
    
    您将看到一个玩家端界面，用于与其他玩家互动。

3. **加入游戏**：
    - 输入您的昵称（建议按座位顺序）。
    - 点击“加入游戏”按钮。
    - 系统会自动将您加入当前游戏房间。
  
4. **等待其他玩家**：
    - 其他玩家也需要加入游戏，系统会自动将他们加入当前游戏房间。
    - 当所有玩家都加入后，游戏开始。

## ⚠️ 注意事项

- **数据存储**：当前版本为**内存运行模式**。**请勿关闭运行 `main.py` 的终端窗口**，否则当前游戏进度会丢失。

## 🛠️ 技术栈

- **后端**: Python 3.8+ + Flask 2.3.0
- **前端**: 原生 HTML5/CSS3/JavaScript ES6+（无复杂框架依赖）
- **渲染引擎**: WebGL 2.0（背景特效）
- **字体**: Cinzel & Noto Serif SC
- **版本控制**: Git（线性提交历史，Rebase 工作流）

### 项目依赖

详见 `requirements.txt`：

```text

Werkzeug==2.3.0
```
